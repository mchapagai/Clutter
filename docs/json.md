# JSON
 > JSON stands for JavaScript Object Notation

Light-weight text-bases designed for human-redable data exchange.

### Uses
- used while writing JavaScript based app that includes browser extension and websites
- user for serializing and transmitting structured data over network connection
- user to transmit data between server and application (mobile, web)

### Charateristics
- easy to read and write
- light-weight text-based format
- language independent
